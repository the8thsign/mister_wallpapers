# MiSTer Wallpapers

A curated collection of high-quality **4:3 wallpapers** designed specifically for **MiSTer FPGA**.

These wallpapers are optimized for 4x3 aspect ratio while preserving the original artwork whenever possible.

---

## Installation

To add this wallpaper collection to the MiSTer Downloader, edit your `downloader.ini` file and add the following:

```ini

[the8thsign/mister_wallpapers]

db_url = https://raw.githubusercontent.com/the8thsign/mister_wallpapers/db/db.json.zip

```

Save the file and run **update_all.sh** on your MiSTer.
